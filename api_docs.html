Авторизация

Все запросы API должны содержать API-токен

Все запросы API должны содержать HTTP-заголовок Accept: application/json

Все успешные запросы API возвращают HTTP-статус 200

Передача API-токена возможна двумя способами:

1. Предача API-токена в GET парамете token, например:

2
3
4
GET /api/v1/offer?offset=0&limit=1000&token
    =2ltgXQM64o4pa66W5tkVy8EcGUiEwlcE8oJJdA80fnYhPhpGiYLXPVRPOiaRN7Jj HTTP/1.1
Host: my.saleads.pro
Accept: application/json
